Stop hook stdin JSON schema (Claude Code 2.x):
```json
{
  "session_id": "abc123",
  "transcript_path": "/path/to/transcript.jsonl",

  "permission_mode": "default",
  "hook_event_name": "Stop",
```

```json
{"decision": "block", "reason": "Write your session report to /var/log/claude-secure/spool.md ..."}
```

- Silent: exit 0 (no stdout)
- Or explicit JSON: `{"hookSpecificOutput": {"hookEventName": "Stop", "additionalContext": "..."}}`
