Rewrite proxy/proxy.js to add secret redaction, placeholder restoration, and auth forwarding. The proxy remains a single file using only Node.js stdlib (http, https, fs). Keep the existing buffered proxy pattern and add transform steps.

**Constants and state (top of file):**
- `const fs = require('fs');` (add to existing http/https requires)
- `const UPSTREAM = process.env.REAL_ANTHROPIC_BASE_URL || 'https://api.anthropic.com';`
- `const WHITELIST_PATH = process.env.WHITELIST_PATH || '/etc/claude-secure/whitelist.json';`
- `const warnedEnvVars = new Set();` -- track which missing env vars have been warned about (per D-02)

**Function: loadWhitelist()**
- `fs.readFileSync(WHITELIST_PATH, 'utf8')` wrapped in try/catch (per D-01, SECR-04)
- `JSON.parse(raw)` also in try/catch
- On failure: `console.error('Failed to load whitelist: ...')` and return `null`
- Caller returns 500 JSON response `{"error":"Proxy configuration error","detail":"..."}` when null

**Function: buildMaps(config)**
- Iterate `config.secrets || []`
- For each entry: `const realValue = process.env[entry.env_var];`
- If `!realValue`: check `warnedEnvVars.has(entry.env_var)`, if not warn via `console.warn('Secret env var not set: ' + entry.env_var)` and add to set (per D-02). Then `continue;`
- Push `[realValue, entry.placeholder]` to `redactMap` array
- Push `[entry.placeholder, realValue]` to `restoreMap` array
- Sort both arrays by `(a, b) => b[0].length - a[0].length` (longest search string first to prevent partial match corruption)
- Return `{ redactMap, restoreMap }`

**Function: applyReplacements(text, pairs)**
- If `!text` return `text` (handles empty bodies per Pitfall 6)
- `let result = text;`
- `for (const [search, replace] of pairs) { result = result.replaceAll(search, replace); }` (per D-03, D-06)
- Return `result`

**Function: prepareHeaders(incomingHeaders, bodyByteLength)**
- Shallow copy: `const headers = { ...incomingHeaders };`
- Delete Claude container's auth headers (per D-08): `delete headers['x-api-key']; delete headers['authorization']; delete headers['anthropic-api-key'];`
- Strip `accept-encoding` to prevent compressed responses from Anthropic (per Research Open Question 2): `delete headers['accept-encoding'];`
- Set proxy's own auth (per D-09, OAuth preferred):
  ```
  const oauthToken = process.env.CLAUDE_CODE_OAUTH_TOKEN;
  const apiKey = process.env.ANTHROPIC_API_KEY;
  if (oauthToken) { headers['authorization'] = 'Bearer ' + oauthToken; }
  else if (apiKey) { headers['x-api-key'] = apiKey; }
  ```
- Set `headers['content-length'] = String(bodyByteLength);` (per D-10)
- Return `headers`

**Request handler (http.createServer callback):**
1. Buffer request body: `let body = ''; req.on('data', chunk => { body += chunk; }); req.on('end', () => { ... });`
2. Load config: `const config = loadWhitelist();` -- if null, respond 500 and return
3. Build maps: `const { redactMap, restoreMap } = buildMaps(config);`
4. Redact outbound body: `const redactedBody = applyReplacements(body, redactMap);` (SECR-02)
5. Parse upstream URL: `const url = new URL(req.url, UPSTREAM);`
6. Prepare headers: `const headers = prepareHeaders(req.headers, Buffer.byteLength(redactedBody));` then `headers['host'] = url.host;`
7. Make upstream request via `https.request({ hostname: url.hostname, port: 443, path: url.pathname + url.search, method: req.method, headers })`
8. Buffer upstream response: `let responseBody = ''; upstreamRes.on('data', chunk => { responseBody += chunk; });`
9. On upstream response end: restore placeholders: `const restoredBody = applyReplacements(responseBody, restoreMap);` (SECR-03)
10. Build response headers: `const resHeaders = { ...upstreamRes.headers }; resHeaders['content-length'] = String(Buffer.byteLength(restoredBody)); delete resHeaders['transfer-encoding'];` (remove chunked encoding, per D-10)
11. `res.writeHead(upstreamRes.statusCode, resHeaders); res.end(restoredBody);`
12. On upstream error: `res.writeHead(502, {'content-type':'application/json'}); res.end(JSON.stringify({error:'Bad Gateway',detail:err.message}));`

**Server listen (unchanged):**
- `const PORT = process.env.PROXY_PORT || 8080;`
- `server.listen(PORT, '0.0.0.0', () => { console.log('Proxy listening on :' + PORT); });`

**Log on startup:** Print `console.log('Whitelist path: ' + WHITELIST_PATH);` and whether auth is configured: `console.log('Auth: ' + (process.env.CLAUDE_CODE_OAUTH_TOKEN ? 'OAuth' : process.env.ANTHROPIC_API_KEY ? 'API key' : 'NONE'));`
